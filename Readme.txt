    Социальная сеть.

1. создать окно приветсвия с возможностью регистрации нового пользователя.
2. Все пользователи сети должны быть зарегистрированы. Иметь уникальный логин и пароль.
3. Зарегистрированный пользователь должен иметь возможность сразу попадать на свою страницу.
4. Создать возможность добавлять на свою страницу файлы (фотографии, музыку, сообщения).
5.Создать возможность обмена файлами (сообщениями)  между пользователями
6. Создать возможность поиска пользователей по логину, имени, номеру телефона.
7. У каждого пользователя должен быть уникальный адрес эл почты (попытка не допустить одного чела с несколькими логинами). Осуществлять проверку почты
    вновь зарегистрированных.
8. Желательно осуществлять проверку телефонов.
9. Каждый пользователь может приглашать других пользователей, обмен сообщениями между пользователями осуществляется если пользователи приняли приглашения друг друга.
